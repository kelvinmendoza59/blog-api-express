# Blog API

Simple blog API with Express and planned React frontend.

## Tech Stack

**Backend:**
- Node.js
- Express
- JSON file storage

**Frontend (planned):**
- React
- Axios

## Installation

```bash
npm install
npm start
```

## API Endpoints

- GET /api/posts - List all posts
- GET /api/posts/:id - Get post by ID
- POST /api/posts - Create new post
